# Privacy Policy for Markdown Viewer Extension

**Effective Date**: February 6, 2026

## Data Collection

Markdown Viewer does not collect, store, or transmit any user data. All markdown rendering happens locally in your browser.

## No Tracking

We do not use analytics, tracking, or any data collection tools.

## No Network Requests

The extension works completely offline and makes no network requests (except for external images embedded in your markdown files, which are loaded directly by your browser).

## Third-Party Services

The extension does not use any third-party services or APIs.

## Third-Party Libraries

The extension uses the following open-source libraries for local processing:
- marked.js - For markdown parsing
- Prism.js - For syntax highlighting
- GitHub Markdown CSS - For styling

All processing happens locally in your browser. These libraries do not collect or transmit any data.

## Local Storage

The extension stores your theme preference (light/dark mode) in your browser's local storage. This data never leaves your device.

## Permissions

This extension requires access to file URLs (file://) to render local markdown files. No other permissions are requested.

## Changes to This Policy

Any changes to this privacy policy will be posted in this document and in the GitHub repository.
