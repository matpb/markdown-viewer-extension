üöÄ HOW TO USE

1. Install the extension
2. Go to chrome://extensions/ and find "Markdown Viewer"
3. Click "Details" and enable "Allow access to file URLs" (important!)
4. Open any .md file in Chrome, or select text on any webpage and click the extension icon
5. Watch it transform into beautiful HTML!

That's it! No configuration needed.
